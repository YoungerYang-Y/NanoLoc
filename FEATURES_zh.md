# NanoLoc 功能详解

NanoLoc 是一个全方位的本地化管理平台。以下是其核心功能的详细介绍。

## 1. 项目管理
- **多项目支持**：同时管理多个本地化项目的翻译工作。
- **语言配置**：
  - **基础语言**：选择源语言（例如：英语）。
  - **目标语言**：从标准语言列表中选择，UI 自动格式化显示（英文名 + 本地名 + 代码）。
- **AI 设置**：按项目配置 AI 提供商（兼容 OpenAI 接口），或使用全局默认设置。
- **访问控制**：支持公开注册与安全的身份验证（邮箱/密码）。

## 2. 本地化工作流
### 词条管理
- **增删改查 (CRUD)**：创建、读取、更新和删除翻译 Key。
- **行内编辑**：直接在表格中双击即可编辑内容。
- **备注 (Remarks)**：为译者添加上下文或注释。支持通过 Tooltip 查看长文本。
- **审计追踪**：记录每个单元格的“最后修改人”，确保责任可追溯。

### 翻译操作
- **单元格翻译**：点击任意单元格上的“魔术棒”图标，利用 AI 仅翻译该项。
- **行翻译**：一键翻译整行（所有缺失的目标语言）。
- **列翻译**：通过列标题一键翻译特定语言列（所有缺失项）。
- **批量翻译**：强大的全局工具，扫描整个项目中缺失的翻译，并在后台分批处理（每批 20 条）。
- **智能填充**：“Copy English to All” 功能可快速填充占位符或未翻译的术语。

### 全局状态指示
- **实时反馈**：当*任何*翻译操作正在进行时，屏幕顶部会出现一个醒目的全宽“Translating...”橫幅，防止冲突操作。

## 3. 数据集成
### 导入 (Import)
- **Android 支持**：原生支持导入 `strings.xml` 文件。
- **冲突解决**：
  - 自动检测现有 Key。
  - 使用新内容更新 Value。
  - 自动将旧值归档到“Remarks”字段以供参考。
  - 提供详细的导入统计（新增、更新、跳过）。

### 导出 (Export)
- **CSV 导出**：生成包含所有词条的标准 CSV 文件。
  - **Excel 兼容**：包含 UTF-8 BOM，确保在 Excel 中正确显示特殊字符。
  - **安全文件名**：文件名经过清理，防止系统错误。
  - **纯净输出**：导出文件中不包含“Remarks”列，适合分发。

## 4. 用户界面 (UI)
- **现代仪表盘**：基于 Shadcn UI 和 Tailwind CSS 构建，提供优质的视觉体验。
- **冻结列**：“Actions”、“Key”和“Remarks”列在水平滚动时保持可见。
- **高级分页**：优化的服务端分页（默认 50 条/页），高效处理大数据集。
- **搜索**：按 Key、Value 或 Remarks 进行实时过滤。
- **响应式头部**：包含用户资料和注销功能的全局导航栏。

## 5. 技术亮点
- **快速且可扩展**：由 Next.js 15 App Router 和 Prisma 驱动。
- **类型安全**：端到端全链路 TypeScript 集成。
- **健壮的错误处理**：API（导入/导出/翻译）具备全面的错误捕获机制。
